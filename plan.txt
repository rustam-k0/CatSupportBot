```markdown

Система для автоматизированного сбора и учёта финансовых операций (донаты и расходы) через Telegram-бота с использованием OCR.


Функциональность
* Telegram-бот для приёма скриншотов финансовых документов
* OCR-распознавание ключевых полей: дата, сумма, комментарий
* Автоматическая классификация операций (доход/расход)
* Ручная корректировка данных через бота
* Экспорт в Google Sheets с фильтрацией по животным

Технологический стек
```
Frontend: Telegram Bot API
Backend: Python (FastAPI)
OCR: Tesseract (локально) / Google Vision AI
База данных: SQLite (MVP) → PostgreSQL
Хостинг: Heroku/Google Cloud Run
```

Структура данных (MVP)
```sql
transactions:
* id (PK)
* animal_name
* transaction_date
* amount
* transaction_type
* comment
* sourcedocumenturl
* ocr_status
* created_at
```

Механика работы

1. Пользователь отправляет скриншот чека в бота
2. OCR извлекает: дату, сумму, комментарий
3. Автоклассификация типа операции (отрицательная сумма = расход)
4. Данные сохраняются в БД с возможностью ручной корректировки
5. Экспорт в таблицу с группировкой по животным


Расширенная версия - Полная автоматизация

Дополнительный функционал
* Интеллектуальное распознавание клиник, процедур, валют
* Автоматический расчёт сальдо после каждой операции
* Мультипроектная поддержка через `project_id`
* Веб-интерфейс для управления и корректировок
* REST API для интеграций
* Умное категоризирование расходов/доходов
